# Tools

Optional tools for one-click local practice.

## ngrok

If the teacher asks for a public URL, download `ngrok.exe` for Windows and place it in this folder:

```text
student-kit/tools/ngrok.exe
```

Then run:

```text
run-with-ngrok.bat
```

Most students do not need this. Use it only when a public webhook or public preview URL is required.
