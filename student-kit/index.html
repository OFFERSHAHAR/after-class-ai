After Class AI - ערכת חניך

זהו אזור תרגול מקומי. כאן אפשר לשמור תבניות n8n מהשיעור, לייבא אותן ל-n8n, ולבנות גרסאות אישיות גם אחרי שהכיתה הסתיימה.

תבניות מהשיעור

שמור workflows שהמורה שלח בתוך templates/.

העבודות שלי

שמור גרסאות אישיות בתוך my-workflows/.

הערות וניסויים

שמור payloads, תובנות ותקלות בתוך notes/.

איך ממשיכים לתרגל

  1. מורידים תבנית workflow מהשיעור או מהממשק.
  2. שומרים אותה בתיקיית templates/.
  3. פותחים n8n ומייבאים את הקובץ.
  4. שומרים עותק אישי ומשנים אותו בלי לפגוע בתבנית המקורית.
  5. אם יש שאלה, שומרים צילום מסך או הערה בתיקיית notes/.
