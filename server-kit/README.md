# After Class AI - Server Kit

ערכת הפעלה למחשב המורה או המחשב שמשמש כשרת הכיתה.

המטרה:

```text
לחיצה אחת -> הממשק עולה -> n8n עולה -> תלמידים נכנסים דרך הרשת
```

## מה הערכה מריצה

1. שרת קבצים מקומי לממשק:

```text
http://SERVER_IP:5174/after-class-ai/
```

2. סביבת:

```text
n8n
```

בכתובת:

```text
http://SERVER_IP:5678/
```

## איך מפעילים

מהמחשב של המורה:

```text
start-class-server.bat
```

הקובץ מפעיל בפנים:

```text
run-ui-server.bat
```

וגם:

```text
run-n8n-server.bat
```

## מה התלמידים פותחים

את הממשק:

```text
http://10.0.0.1:5174/after-class-ai/
```

את סביבת העבודה:

```text
http://10.0.0.1:5678/
```

## דרישות

צריך:

```text
Node.js
```

צריך גם:

```text
n8n
```

אם:

```text
n8n
```

לא מותקן, הסקריפט ינסה להריץ דרך:

```text
npx
```

## הערה חשובה

התלמידים לא צריכים להריץ:

```text
n8n
```

על המחשב שלהם.

הם רק פותחים דפדפן ונכנסים לשרת שלך.
