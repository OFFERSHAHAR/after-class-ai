# בדיקת צאט בוט אוטומטית

## המטרה

התלמיד לא מפעיל פקודות.

התלמיד לא פותח:

```text
CMD
```

התלמיד מתקן את התרגיל בתוך:

```text
n8n
```

ואז חוזר לממשק ולוחץ:

```text
בדוק צאט בוט
```

## מה קורה מאחורי הקלעים

הממשק שולח בקשה אל:

```text
Apps Script
```

הסקריפט שולח בקשה אל:

```text
n8n
```

דרך כתובת:

```text
Webhook
```

התשובה חוזרת לממשק.

במקביל נרשמת שורה בלשונית:

```text
runs
```

אם הצליח:

```text
success
```

אם נכשל:

```text
failed
```

## תנאי חשוב

בתוך:

```text
n8n
```

התרגיל צריך להיות פעיל.

לא מצב:

```text
Listen for test event
```

אלא תהליך פעיל שמקבל כתובת:

```text
/webhook/after-class-support-chat
```

## זרימת תלמיד

1. תלמיד מצטרף לממשק.
2. תלמיד מוריד תרגיל.
3. תלמיד מייבא לתוך:

```text
n8n
```

4. תלמיד משנה את:

```text
Student Edit Area
```

5. תלמיד מפעיל את התהליך.
6. תלמיד חוזר לממשק.
7. תלמיד לוחץ:

```text
בדוק צאט בוט
```

8. הממשק מציג את התשובה.
9. המורה רואה הצלחה או כשלון בלוח.
