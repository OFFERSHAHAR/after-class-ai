# הפעלה בלחיצה אחת לשיעור

## מה עושים מחר

פותחים את התיקייה:

```text
classroom-run
```

ולוחצים על:

```text
START_CLASS_TOMORROW.bat
```

הקובץ מפעיל:

- ממשק כיתה
- סביבת עבודה
- מנהרות ציבוריות אם הכלי נמצא

## מה שולחים לתלמידים

אם כולם באותה רשת:

```text
http://10.0.0.1:5174/after-class-ai/
```

אם לא כולם באותה רשת:

מריצים:

```text
SHOW_PUBLIC_LINKS.bat
```

ושולחים את הקישור שמופיע תחת:

```text
Class interface
```

## מה שולחים כדי לעבוד בסביבת העבודה

אם כולם באותה רשת:

```text
http://10.0.0.1:5678/
```

אם לא כולם באותה רשת:

שולחים את הקישור שמופיע תחת:

```text
n8n workspace
```

## מה התלמיד עושה

1. פותח את קישור הממשק.
2. נכנס למסך תלמיד.
3. כותב שם.
4. כותב קוד שיעור:

```text
AI-203
```

5. לוחץ הצטרפות.
6. לוחץ הורדת תרגיל.
7. פותח את סביבת העבודה.
8. מייבא את התרגיל.
9. משנה רק את האזור המסומן.
10. מריץ.

## מה המורה עושה

1. משתף את לוח המורה.
2. בונה תהליך בהדגמה.
3. מייצא תרגיל אם צריך.
4. שם אותו בשם:

```text
current-class-workflow.json
```

בתיקייה:

```text
workflows
```

5. התלמידים מורידים את התרגיל המעודכן.

## מה לא עושים

לא מבקשים מתלמיד לפתוח:

```text
CMD
```

לא מבקשים ממנו להריץ פקודות.

לא מבקשים ממנו להתקין סביבת עבודה.
