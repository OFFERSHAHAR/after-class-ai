# זרימת עבודה אמיתית לשיעור

## המטרה

להפוך תהליך שהמורה בונה בתוך:

```text
n8n
```

לתרגיל כיתתי שכל תלמיד יכול להוריד, להריץ, לשנות, ולדווח עליו בלוח המורה.

בלי:

```text
CMD
```

בלי פקודות.

בלי מנויים.

בלי התקנות אצל התלמיד.

## הזרימה בפועל

### שלב 1 - המורה בונה תהליך

המורה פותח את סביבת:

```text
n8n
```

שרצה על שרת הכיתה.

המורה בונה תהליך מול הכיתה.

לדוגמה:

```text
Webhook
```

ואז:

```text
Student Edit Area
```

ואז:

```text
Return Result
```

## שלב 2 - המורה מסמן מה התלמיד צריך לשנות

בתוך התהליך צריך להיות צומת ברור בשם:

```text
Student Edit Area
```

רק שם התלמיד נוגע.

כל שאר הצמתים הם תשתית.

כך התלמיד לומד את החומר בלי לבזבז זמן על חיבורי מערכת.

## שלב 3 - המורה מייצא את התהליך

מתוך:

```text
n8n
```

המורה עושה:

```text
Export workflow
```

ומקבל קובץ:

```text
JSON
```

## שלב 4 - המורה מעלה את התרגיל לכיתה

בגרסה הנוכחית, העלאה לכיתה היא פעולה פשוטה:

פותחים את תיקיית:

```text
workflows
```

אפשר דרך:

```text
server-kit/open-workflows-folder.bat
```

שמים את הקובץ שהמורה ייצא בשם:

```text
current-class-workflow.json
```

זה הקובץ שכפתור ההורדה בממשק יוריד לתלמידים.

## שלב 5 - תלמיד מוריד את התרגיל

התלמיד נכנס לממשק הכיתה.

עובר למסך תלמיד.

לוחץ:

```text
הורד workflow
```

הקובץ יורד למחשב שלו.

## שלב 6 - תלמיד מייבא לתוך n8n

התלמיד פותח:

```text
n8n
```

של שרת הכיתה.

הוא מייבא את הקובץ.

בשלב הבא נרצה כפתור אוטומטי שיפתח לו תרגיל אישי.

כרגע זה ידני, אבל בלי פקודות.

## שלב 7 - תלמיד משנה רק את אזור התרגול

התלמיד מחפש צומת בשם:

```text
Student Edit Area
```

ושם מבצע את המשימה.

לדוגמה:

- לשנות תנאי.
- להוסיף שדה.
- לתקן הודעה.
- להחזיר תשובה אחרת.

התלמיד לא נוגע בצומת הדיווח.

התלמיד לא נוגע בתשתית.

## שלב 8 - תלמיד מריץ

התלמיד לוחץ הרצה בתוך:

```text
n8n
```

אם התהליך בנוי עם דיווח אוטומטי, הלוח מתעדכן לבד.

אם עדיין אין דיווח אוטומטי בתרגיל הזה, התלמיד לוחץ בממשק:

```text
הרץ workflow
```

זו תוכנית גיבוי בלבד.

## מה צריך כדי שזה יהיה מושלם

### עכשיו

הורדת התרגיל הנוכחי כבר עובדת דרך:

```text
current-class-workflow.json
```

### השלב הבא

נוסיף לתבנית:

```text
Report Success
```

וגם:

```text
Class Error Reporter
```

כדי שכל הרצה מתוך:

```text
n8n
```

תופיע אוטומטית בלוח.

## משפט למורה

המורה בונה תהליך אמיתי מול הכיתה, מייצא אותו כתרגיל, והתלמידים מורידים אותו בלחיצה אחת.

התלמידים משנים רק אזור מוגדר בתהליך, מריצים מתוך:

```text
n8n
```

והלוח מציג למורה מי הצליח, מי נכשל, ומה צריך עזרה.
