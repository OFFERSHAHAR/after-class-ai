# שרת כיתה

## המטרה

במקום שכל תלמיד יריץ סביבת עבודה בעצמו, מחשב אחד בכיתה משמש שרת.

המחשב הזה מריץ:

```text
After Class AI
```

וגם:

```text
n8n
```

התלמידים נכנסים דרך הדפדפן.

## איך מפעילים

במחשב המורה או השרת:

```text
server-kit/start-class-server.bat
```

הקובץ פותח שני חלונות:

1. ממשק הכיתה.
2. סביבת:

```text
n8n
```

## קישורים לתלמידים

אם כתובת המחשב היא:

```text
10.0.0.1
```

הממשק:

```text
http://10.0.0.1:5174/after-class-ai/
```

סביבת:

```text
n8n
```

```text
http://10.0.0.1:5678/
```

## איך אור עובד

אור לא מריץ:

```text
n8n
```

אצלו.

הוא פותח:

```text
http://10.0.0.1:5678/
```

ועובד על סביבת:

```text
n8n
```

שרצה אצלך.

## מה קורה בזמן שיעור

1. המורה פותח את שרת הכיתה.
2. התלמידים נכנסים לממשק.
3. התלמידים נכנסים לסביבת:

```text
n8n
```

4. המורה בונה תהליך.
5. התלמידים צופים או עובדים על עותקים.
6. דיווחי הרצה מגיעים ללוח הכיתה.

## מגבלות לגרסה ראשונה

זו סביבת כיתה פתוחה בתוך אותה רשת.

לא לתת בה סודות אמיתיים.

לא לשים מפתחות:

```text
API
```

אמיתיים בלי הרשאות ומכסות.
