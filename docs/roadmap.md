# Roadmap - After Class AI

## שלב 1 - דמו להצגה

- ממשק סטטי
- לוח מורה
- מסך תלמיד
- תבניות תרגול
- תיעוד מוצר

## שלב 2 - n8n מקומי

- Docker Compose עם n8n ו-Postgres
- ייבוא workflow לדוגמה
- תרגול webhook בסיסי
- סביבת credentials דמה

## שלב 3 - חיבור אמיתי מהממשק ל-n8n

- API לקריאת workflows
- API לשכפול workflow מתבנית
- API לקריאת executions
- הצגת סטטוס הרצה בלוח המורה

## שלב 4 - כיתה אמיתית

- יצירת חדר שיעור
- קישור הצטרפות לתלמיד
- רשימת תלמידים פעילים
- כפתור "אני צריך עזרה"
- מסך Live Board לשיתוף בזום

## שלב 5 - למידה ארגונית

- ספריית תרגילים לפי רמות
- תרחישים ארגוניים אמיתיים
- מדדי הצלחה
- סיכום שיעור אוטומטי
- ארכיון workflows והרצות
