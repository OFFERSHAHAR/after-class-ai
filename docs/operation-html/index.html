מה המערכת עושה

After Class AI היא שכבת כיתה מעל סביבת n8n מקומית. המורה מדגים, מפרסם תרגיל, התלמידים מתקנים ומריצים, והמורה מקבל חיווי חי על הצלחות, כשלונות ובקשות עזרה.

פשוט לתלמיד

התלמיד לא נוגע בפקודות. הוא מוריד תרגיל, פותח סביבת עבודה, מתקן, ולוחץ בדיקה.

נוח למורה

המורה רואה מי הצטרף, מי הצליח, מי נכשל ומי צריך עזרה, בלי לעצור את השיעור.

חינמי וגמיש

המערכת בנויה לעבוד עם שרת מקומי, גיליון גוגל, וקישור ציבורי זמני לשיעור.

תרשים ההפעלה

זה המסלול המלא מרגע שהמורה מכין תרגיל ועד שהתלמיד רואה תוצאה.
