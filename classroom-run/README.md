# ערכת הרצה לשיעור מחר

המטרה:

```text
לחיצה אחת אצל המורה
```

ואז:

- הממשק המקומי עולה
- סביבת העבודה עולה
- מנהרות נפתחות אם הכלי קיים
- תלמידים מקבלים קישורים

## הקובץ המרכזי

להפעיל:

```text
START_CLASS_TOMORROW.bat
```

## מה נפתח

ממשק הכיתה:

```text
http://127.0.0.1:5174/after-class-ai/
```

סביבת העבודה:

```text
http://127.0.0.1:5678/
```

אם התלמידים על אותה רשת:

```text
http://10.0.0.1:5174/after-class-ai/
```

וגם:

```text
http://10.0.0.1:5678/
```

אם התלמידים לא על אותה רשת, משתמשים במנהרות.

## כלי מנהרה

צריך לשים את הקובץ:

```text
ngrok.exe
```

כאן:

```text
classroom-run/tools/ngrok.exe
```

ואז הקובץ המרכזי יפתח מנהרה לממשק ומנהרה לסביבת העבודה.

## הגדרת חשבון מנהרה

מריצים פעם אחת:

```text
SETUP_NGROK_AUTH.bat
```

מדביקים את הטוקן.

הטוקן נשמר במחשב ולא נכנס לקוד.

## דומיין קבוע לסביבת העבודה

יש קובץ דוגמה:

```text
local-settings.example.cmd
```

מעתיקים אותו בשם:

```text
local-settings.cmd
```

ומוודאים שיש בו:

```text
CLASSROOM_N8N_PUBLIC_URL
```

וגם:

```text
CLASSROOM_NGROK_DOMAIN
```

## אחרי ההפעלה

להריץ:

```text
SHOW_PUBLIC_LINKS.bat
```

הוא יציג את הקישורים הציבוריים שהמנהרה פתחה.

## אם יש תקלה במנהרה

אם יש שגיאה של מנהרה לא זמינה, סוגרים את חלון המנהרה ומפעילים שוב.

הקישור משתנה בכל הפעלה, לכן צריך לשלוח לתלמידים את הקישור החדש.
