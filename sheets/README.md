# Google Sheets Templates

פתח Google Sheet חדש בשם:

```text
After Class AI - Data
```

או השתמש ב-Sheet שכבר הוגדר לפרויקט:

```text
https://docs.google.com/spreadsheets/d/19GOHx2oq1ZsnkggJ-r-8TGvYvhYle3sAWifAzmL-rwQ_/edit
```

צור לשוניות לפי הקבצים בתיקייה הזו, והעתק לכל לשונית את הכותרות והשורות לדוגמה.

לשוניות:

- `sessions`
- `students`
- `exercises`
- `runs`
- `help_requests`

אחרי יצירת ה-Sheet, פתח Extensions > Apps Script והדבק את הקוד מתוך:

```text
google-apps-script/Code.gs
```
