# Contributing to After Class AI

תודה על הרצון לשתף פעולה.

## איך לעבוד על הפרויקט

1. פותחים branch חדש לשינוי.
2. מריצים את הממשק מקומית.
3. בודקים שהמסכים בעברית ו-RTL לא נשברים.
4. מוסיפים תיעוד קצר אם שיניתם התנהגות או מסך.
5. פותחים Pull Request עם הסבר קצר.

## הרצה מקומית

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

פתיחה בדפדפן:

```text
http://127.0.0.1:5173/
```

## כללי עבודה

- לא מעלים קובץ `.env`.
- לא מעלים API keys או credentials אמיתיים.
- workflows לדוגמה נשמרים תחת `workflows/`.
- מסמכים והסברים נשמרים תחת `docs/`.
- שינויי UI צריכים להיבדק גם במסך צר וגם במסך רחב.

## רעיונות טובים ל-PR ראשון

- מסך Live Board לשיתוף בזום
- מסך יצירת שיעור
- קוד הצטרפות לשיעור
- כפתור "אני צריך עזרה"
- חיבור ראשוני ל-n8n API
